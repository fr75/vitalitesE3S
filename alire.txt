Pour générer les fichiers vitalite.json et zones.geojson :

# Générateur aléatoire de vitalités sur chaque zone pour une plage horaire. Modifier dans le script les zones, la plage, la résolution.
python3 generateur_vitalites.py

# Crée zones.geojson à partir de PolygonesLotsLaVallee.kml
python3 conv_zones_kml_vers_geojson.py

=== Ancien
# Crée zones.geojson à partir de zones.txt
# Nota : format geojson pas tout à fait respecté car la dernière coord du polygone n'est pas identique à la première
python3 conv_zones_vers_geojson.py
===

Tester le script (nécessite connexion internet) :
firefox vitalite.html

Nota, bien vérifier que le nom des zones définis dans le générateur de vitalités correspond aux zones définies dans le kml ou txt
